## Building

**PiWall** — Inline firewall built around NFQUEUE, with a Flask dashboard for monitoring and extension cards (Pi-hole, VPN). Runs on a Raspberry Pi 5.
**quantum** — Active C++ project QUANT IDE.
**network-fingerprinter** — Flask web app that shells out to nmap with a stateful command system for network reconnaissance.
**Riemann zero-search framework** — Not claiming to solve the Riemann Hypothesis. A parallel/distributed computing framework for off-critical-strip zero search, using Fourier transforms to improve the time complexity of the alternating series formula. Distributed via BOINC. In progress toward an arXiv submission.
